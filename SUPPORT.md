# Support

## Getting Help

For usage, bugs, and feature requests:
- open a GitHub issue
- include environment details and reproduction steps
- include expected vs actual behavior

For sensitive topics:
- email theshovonsaha@gmail.com

## Before Opening an Issue

1. Read [README.md](README.md) and [documentation/public/SETUP.md](documentation/public/SETUP.md).
2. Search existing issues.
3. Confirm configuration (`.env`) and model/provider availability.

More details:
- [documentation/public/SUPPORT.md](documentation/public/SUPPORT.md)
